📊 Data & Graphs — Key Vocabulary

📏 ScaleThe numbers on the axis — always check what each unit represents (e.g. each square = 5 students)
ModeThe value that appears MOST often in a data set. A set can have more than one mode!
MedianThe MIDDLE value when data is arranged in order. If two middle values, find their mean.
MeanThe AVERAGE — add all values, then divide by how many there are.
RangeThe SPREAD — subtract the smallest value from the largest. Range = Max − Min.
FrequencyHow many times a value occurs. A frequency table counts each item.
Tally ChartGroups of 5 marks (||||) used to record data as it is collected.
Most/Least PopularHighest/lowest bar (or frequency) in the data.
